Contrastive Loss

used to train a model to pull the digital representations of similar data points closer together in space while pushing dissimilar ones farther apart

by Charles Iliya Krempeaux

The Contrastive Loss is a loss function and objective function used in artificial intelligence (AI) and machine learning (ML).
